# mimic-iv-healthcare-analytics-
End-to-end healthcare analytics portfolio using MIMIC-IV: patient segmentation, readmission risk prediction, and hospital resource optimization with SQL-style feature engineering and Python.

# MIMIC-IV Healthcare Analytics Portfolio

This repository demonstrates end-to-end healthcare data science and analytics using the MIMIC-IV clinical dataset.

## Projects

### 1. Patient Segmentation
Unsupervised learning to segment patients based on clinical behavior and risk patterns.

### 2. 30-Day Readmission Risk Prediction
Supervised ML model to identify patients at high risk of readmission for care prioritization.

### 3. Hospital Resource Optimization & Length of Stay Analysis
Operational analytics to understand ICU utilization, patient flow, and resource-heavy patients.

## Tech Stack
Python, Pandas, SQL-style feature engineering, Jupyter Notebook, Healthcare Analytics
